# 🚀 Obviously.ai - Landing Page Clone

This project is a high-fidelity clone of the **Obviously.ai** landing page. It was built to demonstrate my ability to translate a complex web design into a clean, functional, and responsive website.

### ✨ Features:
- **Hero Section:** Clean and professional layout with high-quality styling.
- **Responsive Layout:** Fully optimized for mobile, tablet, and desktop screens.
- **Modern UI:** Using custom fonts, spacing, and modern CSS techniques.
- **Navigation:** A functional and aesthetic header menu.

### 🛠️ Technologies Used:
- **HTML5:** For semantic and organized structure.
- **CSS3:** For advanced layouts, Flexbox, and custom styling.
- **Google Fonts:** For typography consistency.
